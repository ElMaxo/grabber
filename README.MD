**grabber**

**Сборка сервиса**

`make build`

**Запуск**

Для запуска сервиса требуется Docker с поддержкой swarm. Перед
первым запуском необходимо выполнить `make init` для инициализации окружения.

`make run` запустит сервер Postgres и сервис сбора новостей

**Остановка сервиса**

`make stop`

**Запуск тестов**

`make local-test`

**Работа с сервисом**
